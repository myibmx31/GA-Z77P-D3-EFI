# GA-Z77P-D3 Hackintosh
GA-Z77P-D3 Clover EFI. Perfect for macOS 10.14.6 & 10.15 Beta

| 硬件 | 型号 |
| ----| ---- |
| CPU | Intel Xeon E3 1230 V2 |
| 显卡 | Sapphire RX 560D |
| 声卡 | Realtek ALC 887 |
| 网卡 | Realtek RTL 8111 |
